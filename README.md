# 📊 Sales Performance Dashboard

## 📌 Project Overview

The project involved data preparation, SQL analysis, and the development of an interactive Power BI dashboard to explore sales performance through KPIs, trends, product categories, regions, and top customers

---

## 🚀 Dashboard Features

- KPI Cards (Revenue, Sales, Profit, Orders)
- Monthly Sales Trend
- Sales by Product Category
- Profit by Product Category
- Top 5 Customers by Sales
- Year and Region slicers for interactive filtering

---

## 🛠 Tools & Technologies

- Microsoft Excel
- SQL
- Power BI
- DAX

---

## 📈 Business Insights

- Furniture generated the highest sales and profit.
- Monthly sales fluctuate throughout the year with noticeable peaks.
- The dashboard identifies the top five customers by sales.
- Interactive filters allow users to analyze data by year and region.
